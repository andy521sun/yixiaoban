## 🔧 管理命令

### 启动服务
```bash
# 方法1：使用启动脚本（推荐）
cd yixiaoban_app
./start_dev.sh

# 方法2：手动启动
cd backend && node src/server.js &
cd admin && python3 -m http.server 8080 --directory dist &
```

### 停止服务
```bash
# 查找进程ID
ps aux | grep -E "(node.*server|python.*http.server)"

# 停止进程
kill <后端进程ID> <管理后台进程ID>
```

## 🚨 故障排除

### 常见问题

#### 1. 端口被占用
```bash
# 查看端口占用
sudo lsof -i :3000
sudo lsof -i :8080

# 停止占用进程
sudo kill -9 <进程ID>
```

#### 2. Node.js依赖问题
```bash
cd backend
rm -rf node_modules package-lock.json
npm install
```

#### 3. 服务无法访问
```bash
# 检查服务是否运行
ps aux | grep -E "(node|python)"

# 检查端口监听
netstat -tlnp | grep -E "(3000|8080)"
```

#### 4. API响应错误
```bash
# 查看后端日志
tail -f backend/logs/*.log

# 测试健康检查
curl -v http://localhost:3000/health
```
