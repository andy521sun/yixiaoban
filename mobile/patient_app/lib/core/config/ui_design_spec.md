# 医小伴APP - UI设计规范

## 项目概述
- **应用名称**: 医小伴 (Medical Companion)
- **目标用户**: 需要就医陪诊服务的患者及家属
- **核心功能**: 医院查询、陪诊师预约、在线支付、进度跟踪
- **设计风格**: 医疗健康 + 专业可信 + 温暖关怀

## 设计系统

### 1. 色彩系统 (Color Palette)

#### 主色调 (Primary Colors)
- **主蓝**: `#1A73E8` - 信任、专业、医疗
- **辅助绿**: `#34A853` - 健康、安全、成功
- **强调橙**: `#FB8C00` - 温暖、关怀、提醒

#### 中性色 (Neutral Colors)
- **背景白**: `#FFFFFF` - 主背景
- **卡片灰**: `#F8F9FA` - 卡片背景
- **边框灰**: `#E0E0E0` - 分割线、边框
- **文字黑**: `#202124` - 主要文字
- **文字灰**: `#5F6368` - 次要文字
- **提示灰**: `#9AA0A6` - 提示文字

#### 功能色 (Functional Colors)
- **成功绿**: `#0F9D58` - 成功状态
- **警告黄**: `#F4B400` - 警告状态
- **错误红**: `#DB4437` - 错误状态
- **信息蓝**: `#4285F4` - 信息提示

### 2. 字体系统 (Typography)

#### 字体族 (Font Family)
- **主要字体**: `PingFang SC` (iOS), `Noto Sans SC` (Android), `sans-serif` (Web)
- **代码字体**: `Roboto Mono`, `monospace`

#### 字号层级 (Font Scale)
- **H1**: 24sp - 页面标题
- **H2**: 20sp - 模块标题
- **H3**: 18sp - 卡片标题
- **Body1**: 16sp - 正文
- **Body2**: 14sp - 辅助文字
- **Caption**: 12sp - 说明文字
- **Button**: 16sp - 按钮文字

#### 字重 (Font Weight)
- **Regular**: 400 - 正文
- **Medium**: 500 - 强调文字
- **SemiBold**: 600 - 标题
- **Bold**: 700 - 重要标题

### 3. 间距系统 (Spacing)

#### 基础单位: 8dp
- **XS**: 4dp (0.5x)
- **S**: 8dp (1x)
- **M**: 16dp (2x)
- **L**: 24dp (3x)
- **XL**: 32dp (4x)
- **XXL**: 48dp (6x)

#### 布局间距
- **屏幕边距**: 16dp
- **卡片内边距**: 16dp
- **元素间距**: 8-16dp
- **行高**: 1.5倍字号

### 4. 圆角系统 (Border Radius)
- **无圆角**: 0dp - 输入框
- **小圆角**: 4dp - 按钮、标签
- **中圆角**: 8dp - 卡片
- **大圆角**: 16dp - 头像、大卡片
- **全圆角**: 999dp - 圆形元素

### 5. 阴影系统 (Elevation)
- **Level 0**: 无阴影 - 背景层
- **Level 1**: 0 1px 3px rgba(0,0,0,0.12) - 卡片
- **Level 2**: 0 3px 6px rgba(0,0,0,0.16) - 悬浮卡片
- **Level 3**: 0 10px 20px rgba(0,0,0,0.19) - 对话框
- **Level 4**: 0 14px 28px rgba(0,0,0,0.25) - 底部弹窗

## 组件规范

### 1. 按钮 (Buttons)

#### 主要按钮 (Primary Button)
- **背景色**: 主蓝 `#1A73E8`
- **文字色**: 白色 `#FFFFFF`
- **圆角**: 8dp
- **高度**: 48dp
- **内边距**: 水平16dp
- **状态**: 正常、按下、禁用

#### 次要按钮 (Secondary Button)
- **背景色**: 透明
- **边框**: 1dp 主蓝 `#1A73E8`
- **文字色**: 主蓝 `#1A73E8`
- **圆角**: 8dp
- **高度**: 48dp

#### 文字按钮 (Text Button)
- **背景色**: 透明
- **文字色**: 主蓝 `#1A73E8`
- **高度**: 48dp
- **无边框、无背景**

### 2. 输入框 (Text Fields)

#### 标准输入框
- **高度**: 56dp
- **标签**: 上浮标签
- **边框**: 1dp 边框灰 `#E0E0E0`
- **圆角**: 4dp
- **状态**: 正常、聚焦、错误、禁用

#### 搜索框
- **高度**: 48dp
- **图标**: 左侧搜索图标
- **背景**: 卡片灰 `#F8F9FA`
- **圆角**: 24dp

### 3. 卡片 (Cards)

#### 信息卡片
- **背景**: 白色 `#FFFFFF`
- **圆角**: 12dp
- **阴影**: Level 1
- **内边距**: 16dp
- **间距**: 底部16dp

#### 医院卡片
- **特色**: 医院图片 + 基本信息
- **操作**: 预约按钮
- **状态**: 可预约、已满

#### 陪诊师卡片
- **特色**: 头像 + 评分 + 服务次数
- **标签**: 专业领域、语言能力
- **操作**: 选择按钮

### 4. 导航 (Navigation)

#### 底部导航栏
- **高度**: 56dp
- **图标**: 24x24dp
- **文字**: 12sp
- **选中色**: 主蓝 `#1A73E8`
- **未选中色**: 文字灰 `#5F6368`

#### 顶部导航栏
- **高度**: 56dp
- **背景**: 白色 `#FFFFFF`
- **标题**: 居中或居左
- **操作**: 右侧图标按钮

### 5. 列表 (Lists)

#### 单行列表
- **高度**: 56dp
- **图标**: 左侧24x24dp
- **文字**: 主标题 + 副标题
- **操作**: 右侧箭头或开关

#### 双行列表
- **高度**: 72dp
- **主标题**: 16sp 文字黑
- **副标题**: 14sp 文字灰
- **间距**: 垂直8dp

## 页面设计

### 1. 启动页 (Splash Screen)
- **Logo**: 居中显示
- **背景**: 主蓝渐变
- **时长**: 2秒
- **过渡**: 淡入淡出

### 2. 登录/注册页 (Auth Pages)
- **表单**: 手机号 + 密码/验证码
- **验证**: 实时验证提示
- **社交登录**: 微信、支付宝
- **隐私协议**: 底部链接

### 3. 首页 (Home Page)
- **顶部**: 搜索框 + 消息通知
- **轮播**: 医院/服务推荐
- **分类**: 服务类型网格
- **推荐**: 热门医院/陪诊师
- **底部**: 固定导航栏

### 4. 医院详情页 (Hospital Detail)
- **头部**: 医院图片 + 基本信息
- **标签**: 三甲、专科、距离
- **科室**: 可折叠科室列表
- **医生**: 推荐医生轮播
- **预约**: 底部固定按钮

### 5. 陪诊师详情页 (Companion Detail)
- **头部**: 头像 + 基本信息
- **评分**: 星级 + 评价数量
- **标签**: 专业领域、语言
- **服务记录**: 历史服务列表
- **预约**: 时间选择 + 确认

### 6. 订单页 (Order Page)
- **进度**: 订单状态时间轴
- **详情**: 服务信息卡片
- **支付**: 支付方式选择
- **操作**: 取消、联系、支付

### 7. 个人中心 (Profile Page)
- **头部**: 用户信息卡片
- **菜单**: 设置项列表
- **订单**: 我的订单入口
- **帮助**: 客服、反馈

## 交互规范

### 1. 加载状态 (Loading States)
- **页面加载**: 骨架屏
- **内容加载**: 进度条
- **按钮加载**: 旋转图标 + 禁用
- **图片加载**: 占位图 + 渐显

### 2. 空状态 (Empty States)
- **无数据**: 插画 + 提示文字
- **无网络**: 图标 + 重试按钮
- **无权限**: 提示 + 设置入口

### 3. 错误状态 (Error States)
- **表单错误**: 红色边框 + 提示文字
- **网络错误**: 提示 + 重试按钮
- **服务器错误**: 友好提示 + 联系客服

### 4. 成功状态 (Success States)
- **操作成功**: 绿色提示 + 图标
- **支付成功**: 确认页面 + 返回按钮
- **预约成功**: 订单详情 + 分享

## 动效规范

### 1. 转场动画 (Transitions)
- **页面切换**: 水平滑动
- **弹窗出现**: 从底部上滑
- **菜单展开**: 淡入 + 缩放

### 2. 微交互 (Micro-interactions)
- **按钮点击**: 缩放效果
- **列表项点击**: 涟漪效果
- **开关切换**: 滑动效果

### 3. 加载动画 (Loading Animations)
- **旋转加载**: 无限旋转
- **进度加载**: 线性进度条
- **骨架屏**: 闪烁动画

## 无障碍设计

### 1. 视觉无障碍
- **对比度**: 文字与背景至少4.5:1
- **字体大小**: 支持系统字体缩放
- **颜色无关**: 不用颜色作为唯一信息

### 2. 操作无障碍
- **触摸目标**: 最小44x44dp
- **键盘导航**: 支持Tab键导航
- **语音朗读**: 支持屏幕阅读器

### 3. 认知无障碍
- **简单语言**: 避免专业术语
- **明确标签**: 表单字段明确标签
- **错误提示**: 明确错误原因和解决方法
